# library-app

A Libraries App developed to learn about Ember.JS and Firebase.

## Installation

* `git clone <repository-url>` this repository
* `cd library-app`
* `npm install`

## Running / Development

* `ember serve`
* Visit your app at [http://localhost:4200](http://localhost:4200).

### Running Tests

* `ember test`
* `ember test --server`
